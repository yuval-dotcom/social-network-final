# סיכום דרישות הפרויקט

## מטרת המערכת

בניית רשת חברתית מלאה ומתפקדת, כולל משתמשים, קבוצות, פוסטים, הרשאות, צ'אט, חיפושים וגרפים דינמיים.

## דרישות טכניות מרכזיות

- שרת מבוסס Node.js ו-Express.
- אחסון ושליפה מתוך MongoDB.
- צד לקוח מבוסס React.
- הפרדה בארכיטקטורת MVC.
- לפחות 3 מודלים שונים.
- לכל מודל נדרש: Create, Update, Delete, List, Search.
- לפחות 2 חיפושים מתקדמים דרך ממשק המשתמש, כל אחד עם לפחות 3 פרמטרים.
- הרשאות לפי שם משתמש וסיסמה.
- מנהלי קבוצות עם יכולות ניהול מורחבות.
- פיד אישי למשתמש.
- צ'אט בין משתמשים עם Socket.io או WebSockets.
- שימוש ב-jQuery וב-Ajax בצד הלקוח.
- שימוש ב-Video וב-Canvas.
- שימוש ב-CSS3: text-shadow, transition, multiple-columns, font-face, border-radius.
- לפחות 2 גרפים דינמיים עם D3.js, כאשר הנתונים מגיעים מה-DB.
- טיפול בשגיאות, ולידציות ומקרי קצה בצד שרת ובצד לקוח.
